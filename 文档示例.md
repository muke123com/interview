## 搜狗注册接口

**简要描述：**

- sogou注册接口

**请求URL：**

- `/api/bo/register`

**请求方式：**

- POST

**Headers**

- Content-Type: application/json

**请求示例**

```json
{    
    "province": "北京市",    
    "city": "北京市",    
    "companyName": "北京搜狗网络技术有限公司",    
    "contactName": "王凯民",    
    "cellPhone": "17600126088",    
    "registerSource": "index",    
    "source":"quandashi"
}
```

**请求参数：**

| 参数名         | 必选 | 类型   | 说明                               |
| -------------- | ---- | ------ | ---------------------------------- |
| province       | 是   | string | 省                                 |
| city           | 是   | string | 市                                 |
| companyName    | 是   | string | 公司名称                           |
| cellPhone      | 是   | string | 手机号                             |
| registerSource | 是   | string | 注册子来源，index，一般为页面位置  |
| source         | 是   | string | 注册主来源，由搜狗提供，详见README |

**返回示例**

```json
{    
	"code": 0,    
    "msg": "成功"
}
```

**返回参数说明**

| 参数名 | 类型   | 说明     |
| ------ | ------ | -------- |
| code   | int    | 返回码   |
| msg    | string | 描述信息 |

**备注**

- 无
